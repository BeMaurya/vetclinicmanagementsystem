Step 1- Install Jdk 1.0.7 with Jre 1.6.0 version software.

Step 2- After install Jdk 1.6.0 software,then set Java bin folder path in system environment variable.

Step 3- Install Back End Application or Oracle 11g in your System.

Step 4- After install Oracle 11g Application, then create database with name cms so you click on createdatabase.bat
file which is located in database folder inside Setup folder.

step 5- After create database then create all required table which is located in database folder inside - Setup folder.

Step 5- Double click on setup.exe , then then your software will run and ready to use.
